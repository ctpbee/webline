# webline

ctpbee提供的界面版本后端实现

### 快速安装

```bash 
# 等待实现
pip3 install ctpbee_webline
```

### 初始化

在你的项目目录文件下面使用以下命令

```bash
ctpbee_webline initdb
```

此命令会创建默认用户`admin`以及密码`123456`

然后运行你的`ctpbee`的`app`即可
